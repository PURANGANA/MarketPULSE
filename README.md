MarketPULSE: Real-Time Stock Insights, Trends & Alerts is a web-based intelligent stock market prediction
application developed to provide users with real-time market insights, trend analysis, and next-day stock 
movement predictions. The system leverages a deep learning model based on a stacked LSTM architecture 
enhanced with Batch Normalization and Dropout layers to improve prediction accuracy while minimizing 
overfitting. The model is trained using historical stock data along with important technical indicators such as 
Relative Strength Index (RSI), Moving Average Convergence Divergence (MACD), Stochastic Oscillator, trading 
volume, and daily returns. These features enable the model to learn market behavior more effectively than relying 
solely on historical closing prices. The application follows a systematic workflow beginning with the collection of 
historical stock market data using the Yahoo Finance API. The collected data undergoes preprocessing, feature 
engineering, and normalization before being supplied to the LSTM model for training and prediction. The trained 
model forecasts the probability of whether a stock's closing price will increase or decrease in the next trading 
session. In addition, the system provides interactive visualizations, market trend analysis, and real-time alerts, 
enabling users to monitor stock performance efficiently.
